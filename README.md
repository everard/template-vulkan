# DESCRIPTION
This repository contains a template for building Vulkan projects. It provides
simple wrappers for common Vulkan functionality, and SDL2 application skeleton.

Requirements:

* Vulkan-Loader and SDL2 libraries.
* C++23-capable compiler.
* pkg-config and make.

To build application skeleton, simply run `make`.
